## 3. Presentation Sizes

| Format | Dimensions | Pixels (at 96 DPI) | When to Use |
|--------|-----------|-------------------|-------------|
| Widescreen (16:9) | 33.87 x 19.05 cm | 1280 x 720 | Standard — use by default |
| HD Widescreen | 33.87 x 19.05 cm | 1920 x 1080 | When projecting on large screens |
| Standard (4:3) | 25.4 x 19.05 cm | 960 x 720 | Legacy — only if required |
| A4 Landscape | 29.7 x 21.0 cm | 1122 x 794 | Document-style presentations |

### Presentation Typography Rules

| Element | Minimum Size | Recommended Size |
|---------|-------------|-----------------|
| Title (title slide) | 36pt | 44-54pt |
| Section Header | 28pt | 32-40pt |
| Slide Title | 24pt | 28-36pt |
| Body Text | 18pt | 20-24pt |
| Footnotes/Sources | 10pt | 12-14pt |
| Data Labels | 10pt | 12-14pt |

### Slide Layout Best Practices

- **Title slide**: Brand name, presentation title, date, speaker name. Logo featured.
- **Section divider**: Bold header, minimal text, brand color background
- **Content slide**: Slide title + max 6 bullet points OR title + visual
- **Data slide**: Chart/graph + insight callout + source note
- **Closing slide**: CTA, contact info, logo centered

| US Tabloid | 279 x 432 | 11 x 17 | Small poster (US) |

### Print-Specific Requirements

**Resolution**: 300 DPI minimum (600 DPI for text-heavy/fine detail)
**Color mode**: CMYK (convert all RGB to CMYK before production)
**Bleed**: 3mm (0.125") on all sides beyond the trim line
**Safe zone**: 5mm (0.2") inside the trim line — keep critical content here
**File format**: PDF/X-1a or PDF/X-4 for professional printing

|----------|-----------|-------------|-------------|--------|

| YouTube Shorts | 1080 x 1920 | 9:16 | 60s | MP4 |

| Facebook Video | 1280 x 720+ | 16:9 or 9:16 | 240 min | MP4 |
| WeChat Video | 1080 x 1920 | 9:16 | 30s (Channels) | MP4 |

### Video Brand Elements

| Element | Specification |
|---------|--------------|
| Title card | 3-5 seconds, brand colors, logo centered, title text |
| Lower third | Bottom-left, 10% from edges, name + title in brand fonts |
| Watermark | Logo in corner, 15-25% opacity, small but legible |
| End card | 5-10 seconds, logo + CTA + website URL + social handles |
| Transition | Simple cuts or dissolves in brand style (avoid cheesy effects) |
| Music | Licensed tracks that match brand mood |
| Subtitles | Brand body font, white with dark outline or background strip |

## 8. File Format Decision Tree

```
What is the deliverable?
│
├── Digital only (web, social, email, screen)
│   ├── Has transparency needed? → PNG
│   ├── Is it a photograph? → JPG (80-90% quality)
│   ├── Is it an icon or logo? → SVG
│   ├── Is it animated? → GIF or MP4
│   └── Is it a document? → PDF (RGB)
│
├── Print production
│   ├── Final artwork for printer → PDF/X-1a or PDF/X-4 (CMYK, 300 DPI)
│   ├── Editable source file → AI, INDD, or PSD
│   ├── Logo for print vendor → EPS or PDF (vector)
│   └── Photo for print → TIFF or high-res JPG (CMYK, 300 DPI)
│
├── Presentation
│   ├── PowerPoint → PPTX
│   ├── Google Slides → Upload PPTX or share link
│   └── Keynote → KEY
│
├── Video
│   ├── Final delivery → MP4 (H.264, AAC audio)
│   ├── High quality archive → MOV (ProRes)
│   └── Animated graphic → GIF or Lottie JSON
│
└── Brand assets package
    ├── Logos → SVG + EPS + PNG (multiple sizes) + PDF
    ├── Colors → JSON config + ASE (Adobe Swatch) + CSS variables
    ├── Fonts → OTF/TTF/WOFF2 files or Google Fonts links
    └── Templates → Native formats (PPTX, DOCX, INDD, Figma link)
```
